## Assignment 2: Custom WordPress Theme
## WordPress Theme Structure and Architecture
WordPress Theme Strcuture & Architecture refers to the collection of files to support functionality on Wordpress. 
These files are differential, such as HTML, Css, Json and PHP. The two required files for theme architecture are styles.css and index.html. 
Style.css stores the metadata, such as the theme name, footer, and the index.html is the template for the HTML. 

## The Loop and Template Tags
The Loop operates in the PHP file to iterates each post to display the content. The template tags are used to display post contents such as the name, data, or title. 
The template tags I used are get_the_thumbnail(), which increases the post image to large. I used get_date() and get_author(), to display the date and author from the post. The get_the_thumbnail was classed under post-image which stores the post pictures.
